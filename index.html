What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  3. is not neccessary CSS saves a lot of work. It can control the layout of multiple web pages all at once
  4. stylesheet are stored on CSS files.

css can be added in HTML document in 3 way

  • Inline - by using the style attribute in side HTML elements
  • Internal - by using a style elements in the head section
  • External - by using a link elements to link to an extrernal CSS file
  • The Most common way to add CSS , is to keep the styles in external CSS files. However , in this tutorial we will use inine and internal styles , because this is easier to demonstrate , and easier for you to try it yourself

When Comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique elements. ID is also special in that you can use a special URL to link directly to an elements and it's used by JavaScript

In CSS, Selectrs are used to target a specific elements or range of elements on a web page. Once an elements has been targeted, a style or set of style can be applied to the elements.

There is a wide range of selectors available.Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
